Usuário Brasileiro

Este programa foi criado para simplificar a identificação das temperaturas críticas do aço, tanto durante o processo de aquecimento para a austenitização quanto durante o resfriamento para a têmpera. Um arquivo exemplo em formato CSV de um aço de baixa liga com microestrutura completamente perlítica (próxima ao eutetóide) está disponível para uso.
Para utilizar o programa, são necessárias estimativas das temperaturas críticas como dados de entrada, as quais podem ser obtidas do Thermocalc ou por meio da análise dos gráficos gerados pelo programa. Este último método permite ajustar as estimativas com base em regressões lineares para obter valores mais precisos.
As opções de linha de comando para salvar foram desativadas, mas podem ser facilmente reativadas conforme a preferência do usuário.
É importante observar atentamente os intervalos nos gráficos, pois estes podem variar entre experimentos e devem ser ajustados no código conforme necessário.

International User: 

This programme has been developed to streamline the process of identifying critical temperatures in steel, both during the heating phase for austenitisation and during cooling for hardening purposes. An illustrative CSV file of a low alloy steel exhibiting a fully pearlitic microstructure (proximate to eutectoid) is at your disposal.
To utilise the programme, estimations of critical temperatures are requisite as input data, attainable either from Thermocalc or through analysis of the graphs generated by the programme. The latter approach affords the ability to refine estimations via linear regressions, thereby yielding more precise values.
Although the command line options for saving have been deactivated, they can be swiftly reactivated to align with user preferences.
It is important to look carefully at the intervals depicted in the graphs, as these may vary across experiments and should be adjusted within the code as necessary.
